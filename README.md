# React + TypeScript + Vite

# Using:

- node: >=18.0.0
- yarn
- Tailwindcss
- Sử dụng react hook form
- Sử dụng UI library material UI

# For Developer:

- Cài thư mục node_module :

```
yarn
```

- Chạy json server :

```
yarn server
```

- Chạy code môi trường dev :

```
yarn dev
```

# ENV

```
    dev: build:dev
```

```
    staging: build:staging
```

```
    prod: build:prod
```
